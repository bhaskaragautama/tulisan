Instalasi

Mendeley dapat diunduh di sini. Untuk dapat menggunakannya, silakan membuat akun terlebih dahulu dan meng-install Mendeley pada komputer. Lakukan instalasi seperti biasa, tidak ada hal spesial atau tricky pada langkah ini. Setelah terpasang, berikut penampakan dari software Mendeley.

Menambah Pustaka

Metadata pustaka yang bersumber dari buku, jurnal, prosiding, dan lain sebagainya disimpan di aplikasi ini. Tidak hanya metadata, dokumen terkait misalnya artikel jurnal juga dapat dimasukkan di sini. Aplikasi ini juga mendukung drag-and-drop dokumen ke dalam aplikasi. Metadata akan dibaca oleh aplikasi secara otomatis. Berikut contoh menambahkan dokumen ke dalam aplikasi.

Setelah dokumen ditambahkan, Mendeley juga bisa digunakan sebagai document viewer dengan melakukan double-click pada dokumen di daftar yang terlihat. Metadata yang diisi oleh sistem secara otomatis (pada bagian kanan), tidak selalu sesuai dengan metadata yang sebenarnya. Untuk itu, perlu dilakukan perbaikan metadata agar sesuai dengan sumber yang akan disitasi.

Pada gambar di atas, bagian kiri menunjukkan metadata di Mendeley (terisi otomatis ketika dokumen ditambahkan). Sedangkan, bagian kanan menunjukkan metadata di web jurnal tempat artikel di-publish. Penomoran menunjukkan hubungan metadata antara Mendeley dengan website. Untuk metadata nomor 5 yaitu halaman, jika tidak ditemukan pada website, biasanya akan ditemukan pada dokumen artikelnya. Contoh ketidaksesuaian terjadi pada metadata nomor 1, 3, dan 6. Untuk itu, metadata pada Mendeley perlu disesuaikan secara manual berdasarkan metadata sumbernya, entah itu website/buku/lain sebagainya. Sumber pustaka juga dapat ditambahkan secara manual untuk menjamin kesesuaian metadata, seperti contoh berikut.

Mendeley - Microsoft Word

Untuk dapat digunakan bersama dengan Microsoft Word, plugin Mendeley harus di-install terlebih dahulu pada Word.

Setelah ter-install, tools Mendeley dapat diakses pada tab References. Berikut adalah tampilan pada Word.

Mendeley - Sitasi dan Daftar Pustaka

Berikut adalah langkah-langkah bagaimana menggunakan Mendeley yang mencakup:

  • Membuat sitasi dalam tulisan, berdasarkan pustaka yang sebelumnya ditambahkan pada aplikasi Mendeley.
  • Membuat daftar pustaka secara otomatis, berdasarkan sitasi yang ditambahkan pada tulisan.
  • Mengubah style sitasi dan daftar pustaka yang digunakan.
